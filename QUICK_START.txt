🐍 CÁCH 2: NÂNG CAO - CRAWL GIÁ REALTIME
═══════════════════════════════════════════════════════════════

BƯỚC 1: Mở Command Prompt (Windows) hoặc Terminal (Mac/Linux)
        ✅ Windows: Nhấn Win+R, gõ "cmd", Enter
        ✅ Mac: Mở Finder → Applications → Utilities → Terminal
        ✅ Linux: Mở Terminal

BƯỚC 2: Chuyển đến thư mục script
        C:\> cd "C:\Users\<YourName>\OneDrive\Documents\Claude\Projects\Tìm mua xe ô tô"

BƯỚC 3: Chọn 1 trong 3 cách:

   OPTION A - Crawl 1 lần:
   $ python crawler_xe_oto.py crawl
   → Sẽ lấy giá từ website & lưu vào car_prices.json
   → Tổn thời gian: ~30 giây

   OPTION B - Khởi động API Server (Tốt nhất):
   $ python crawler_xe_oto.py api
   → Mở cửa sổ dashboard, nhấn "🔄 Cập Nhật Giá"
   → Dashboard sẽ kết nối & tải giá realtime từ API
   → Tổn thời gian: tuỳ ý

   OPTION C - Xem dữ liệu đã lưu:
   $ python crawler_xe_oto.py load
   → Hiển thị giá từ file car_prices.json

💡 MẸO NHANH
═══════════════════════════════════════════════════════════════

🎯 MUỐN CÓ GIÁ REALTIME?
   1. Mở Command Prompt
   2. Gõ: python crawler_xe_oto.py api
   3. Để chạy (đừng đóng cửa sổ)
   4. Mở dashboard, nhấn "🔄 Cập Nhật Giá"

🎯 GIÁ KHÔNG CẬP NHẬT?
   1. Đóng Command Prompt (Ctrl+C)
   2. Chạy lại: python crawler_xe_oto.py api
   3. Thử lại trong dashboard

🎯 MUỐN CHẠY HÀNG NGÀY TỰ ĐỘNG?
   1. Windows: Mở Task Scheduler
   2. Tạo task chạy: python crawler_xe_oto.py crawl
   3. Đặt thời gian: 6:00 AM hàng ngày
